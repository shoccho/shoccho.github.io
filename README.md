# shoccho portfolio

Minimal Hugo portfolio for shoccho.

## Run locally

Install Hugo, then run:

```sh
hugo server
```

Build the static site:

```sh
hugo
```

The generated site will be written to `public/`.

## Content

- `content/work/` contains professional work summaries.
- `content/projects/` contains curated public projects.
- `content/notes/` is reserved for future blog posts and project logs.

The design is intentionally quiet: text-first, no large portrait, no imported theme, and no inflated claims.
